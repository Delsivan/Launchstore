## LaunchStore

Launchstore é um marketplace/e-commerce para compra e venda de produtos ou serviços.
Projeto criado no Bootcamp LaunchBase da Rocketseat.

O projeto foi desenvolvido utilizando as seguintes tecnologias:
- HTML
- CSS
- JavaScript
- Node.js
- Nunjucks
- PostgreSQL
- Express
- NPM
- Nodemon
- Nunjucks
- Nodemailer
